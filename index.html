Marks: grade tracking

Marks is a simple grade tracking application specifically developed to help teachers keep track of grades. The two main aspects driving the development of the application are user data storage and simplicity. Firstly, data is stored locally (on the user's device) to add an extra layer of security for students' data. Secondly, the application has to be easy to use.

Updated: 23-Fed-2020

After initial installation right-click then select open.

Features:
- Add, update and delete for Classes, Students, Subjects, Exams, Grades
- View and download grade charts
- Use the calendar feature to create and save class schedules and events
